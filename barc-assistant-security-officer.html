5. The Skill Tests in Detail üó£Ô∏è

The final stage for most posts is a skill-based test to determine practical suitability.

Security Cadre

  • **Physical Test:** Includes running, long jump, chin-ups/push-ups.
  • **Written Test:** Assesses comprehension and analytical ability.

Admin & Driver Cadre

  • **UDC:** English descriptive paper and computer typing test.
  • **Stenographer:** Shorthand and typing speed tests.
  • **Driver:** Practical driving test for light and heavy vehicles.

6. Roles & Responsibilities üåü

Each role in the administrative and support cadre has distinct and important responsibilities.

Security Staff

  • **ASO:** Supervise security operations, access control, and manage guard shifts.
  • **Security Guard:** Guarding and patrolling sensitive installations.

Clerical Staff

  • **UDC:** File management, drafting letters, maintaining records.
  • **Stenographer:** Taking dictation, typing official documents.

Support Staff

  • **Driver:** Operating official vehicles for personnel and material transport.

Level 1: Qualifying Test

For admin posts, this is a preliminary objective test on general subjects. For security posts, this is the Physical Endurance Test (PET). This stage is for screening/qualification only.

Level 2: Mains / Advanced Test

Candidates who clear Level 1 appear for the main written test. For UDC, it is a descriptive English paper. For security staff, it is an objective test on comprehension and analytical skills. The merit list is often based on this stage.

Level 3: Skill Test

This is the final, qualifying stage. It includes typing tests for UDC, stenography tests for Stenographers, and a practical driving test for Drivers. Security personnel who clear the written test are selected based on its merit.
