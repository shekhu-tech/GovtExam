Written Test Details

This is an objective-type test covering the syllabus mentioned in the notification. It's a screening test to shortlist candidates for the next stage. Marks from this test are often used for final merit list preparation.

Skill / Physical Test

This stage is qualifying in nature. For drivers, it involves a driving test. For Security Guards and Firemen, it includes a Physical Efficiency Test (PET) like running, and a Physical Standards Test (PST) for height, chest, etc.

Document Verification & Medical

Candidates who clear the skill/physical test are called for DV. You must produce all original documents. Following this, a medical examination is conducted to ensure you meet the required medical standards for the post.
